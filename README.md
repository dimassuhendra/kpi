Proyek ini adalah platform berbasis web yang memungkinkan staf IT/TAC untuk melaporkan tugas harian mereka secara mendetail. Manager dapat memvalidasi laporan berdasarkan durasi pengerjaan, tingkat kemandirian, dan inisiatif staf. Sistem ini menghilangkan sistem poin tradisional dan menggantinya dengan verifikasi kualitas aktivitas.

âœ¨ Fitur Utama
Dynamic Mission Logging: Staf dapat menginput aktivitas berdasarkan kategori yang telah dikonfigurasi.

Manager Validation Queue: Antrean validasi laporan yang masuk secara real-time untuk ditinjau oleh Manager.

Smart Analytics (Donut Charts): Visualisasi persentase inisiatif (Temuan Sendiri vs Penugasan) dan kemandirian (Mandiri vs Bantuan).

Activity Categorization: Manajemen kategori aktivitas yang fleksibel tanpa perlu mengatur bobot nilai.

Staff Performance Overview: Pantauan total kasus dan efektivitas staf secara transparan.
